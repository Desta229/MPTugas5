Aplikasi Form Registrasi Android

Deskripsi

Aplikasi ini dibuat untuk memenuhi tugas Week 5.
Berisi form registrasi dengan berbagai komponen UI dan validasi.


Fitur

Input Nama, Email, Password, Confirm Password
Validasi input
Radio Button (Jenis Kelamin)
Checkbox (Hobi minimal 3)
Spinner (Pilihan jurusan)
AlertDialog saat submit
Long Press pada tombol


Teknologi

Kotlin
Android Studio
Material Design
